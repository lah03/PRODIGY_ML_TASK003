## 📌 Project Overview

This project implements a **Support Vector Machine (SVM)** model to classify images of cats and dogs using a dataset provided in CSV format.
The dataset contains extracted image features, and the model is trained to distinguish between the two categories.

---

## 🎯 Objective

* Build a machine learning model using **SVM**
* Perform **data analysis and visualization**
* Evaluate model performance using metrics and charts
* Generate predictions for test data

---

## 📂 Dataset

The dataset consists of:

* `train.csv` → Training data with features and labels
* `test.csv` → Test data for prediction
* `submission.csv` → Output file with predicted labels

### 🏷️ Label Encoding

* `0` → Cat
* `1` → Dog
